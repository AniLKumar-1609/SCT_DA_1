# SCT_DA_1 - Sales Dashboard

## SkillCraft Technology Data Analytics Internship

### Project Objective
Develop an interactive Sales Dashboard using Microsoft Excel to analyze sales performance and visualize business insights.

### Features
- Total Sales Analysis
- Sales by Category
- Sales Trend Over Time
- Interactive Slicers (Region, Category, Segment, Sub-Category)

### Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

### Files Included
- Sales_Dashboard.xlsx
- Dashboard Screenshot

### Outcome
This dashboard enables interactive analysis of sales data and provides valuable business insights through charts and filters.
